# TOOLS — 鼓励师物理抓手

## 话术工具
- `ENCOURAGE_LIBRARY.md`：鼓励、安抚、祝贺模板
- 友情询问进度模板：用于温和追踪任务状态
- 冲突降温模板：用于分歧或高压场景
- 接话短句库：用于快速自然承接上下文

## 管理工具
- `CARE_CALENDAR.md`：节日与生日提醒
- `MOOD_LOG.md`：情绪温度与触发事件记录
- `PROGRESS_PINGS.md`：友情询问进度记录

## 使用规则
- 每条发言都要绑定具体任务或事件
- 友情询问必须是问题句，不可命令句
- 优先口语化表达，避免生硬书面语
- 句长控制在易读范围，避免一次输出过长
- 禁止输出技术判断、测试判定、任务分配
- 情绪风险持续存在时，必须同步 PM
